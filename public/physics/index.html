Portfolio: Wolfram Physics Project

The Project

The Wolfram Physics Project isâ€¦well, that’s what made this a design challenge. This is a theoretical physics framework based off the CEO’s long-standing personal research project into cellular automata. In short…in as much as I can understand it and summarize it…it is an exploration of how complex physical phenomena can evolve from relatively simple initial conditions and computational rules. (The boss summarizes it here. There’s really quite a lot of graph theory.)

The Goals

  • Create a robust design system that allowed easy iteration and quick hand-off to developers.
  • Craft an web-based page interface for a presentation of the ebook of this content.
  • Design a gallery of graphs and models of cellular automata and network systems.
  • Fashion an online file system showing the historical development of this material.

The Constraints

  • The project’s content was technical. Very technical. (I have a perfunctory knowledge of general & special relativity, quantum mechanics, and cosmology, but once you get to causal invariance in graph theory, I shrug epistemically.)
  • …oh, and we started this project in March 2020. As the COVID pandemic began in earnest. That was a constraint for sure. (Fortunately we were already a heavily remote work-centric company.)

The Process

  • The design of the site involved heavy use of in-browser prototyping, using a static site generator, as well as HTML/CSS/JS written by me. This allowed me to iterate rapidly based off evolving and incoming content, and produced deliverables that were immediately ready for our web team to deploy.
    • A key example: CSS grid. Our web team had never utilized CSS grid at all, in any capacity. But, it was extremely useful for several aspects of this site, and by providing CSS to our developers, we rapidly deployed this to the world without significant re-styling to use older techniques.
  • The development of the style and content presentation was a result of many, many meetings with stakeholders, including the CEO and his physics team, which required a rapid uptake of the technical material. And, as noted in the constraints, the bulk of this process took place during the initial wave of the pandemic, which, as one might imagine, necessitated flexiblity with dealing with the interesting nature of the time.
